Core Capabilities

AI that watches so you don't have to

A full pipeline from pixel to alert — no human monitoring required.

🔭

Person Detection

YOLOv8n detects every individual in the scene in real time, counting density across the crowd zone.

🌡️

Density Heatmaps

Gaussian blob kernel estimation overlays a heatmap highlighting dangerous crowd concentration zones.

💨

Motion Analysis

Dense optical flow tracks how chaotically the crowd is moving — a key stampede precursor signal.

🦴

Pose & Fall Detection

MediaPipe Pose identifies fallen individuals — critical indicators of an active stampede event.

⚡

Risk Scoring

A weighted composite model fuses all signals into a single High / Medium / Low risk classification.

📲

Emergency Alerts

On High risk: Twilio sends SMS + voice call to responders with GPS address and nearest police station.

🚨 Instant Emergency Response

When High Risk is detected, the system immediately dispatches both an SMS message and a voice call via Twilio — with the exact GPS address and nearest police station name already included.

📍 GPS reverse-geocoded address via OpenStreetMap
🚔 Nearest police station via Overpass API
💬 Automated SMS with full context
📞 Automated voice call for urgent escalation
